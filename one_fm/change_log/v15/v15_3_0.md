# Version 15.3.0 Release Notes

### ONE FM NEW FEATURES

- Roster V2 - Revamped roster
- Merged Basic and OT Roster views
- New Change Employee Schedule OT option in Roster
- New Search widget in Roster
- Filters now show only active Projects, Sites, Shifts, and Operations Roles in Roster
- Employee filters show matching IDs and names only in Roster
- Pagination added in staff view (default: 50) in Roster
- Reliever selection in Day Off popup changed to Link field  in Roster
- Site supervisor name and project manager name display in Employee record
- Is Unscheduled field added to Attendance doctype
- Roster Day Off Take Action limited to specific roles
- Mark attendance for unscheduled employees,Schedule Mark for Active Employees feature.
- Option to disable user email notifications.
- Configuration to include workflow action buttons in email
- Notification to ticket raiser on task creation and closure with resolution
- Allow resolution details update from Helpdesk

### Bug Fixes

- Post Checker now validates against Project Expected Date.
- Removed flush as Frappe handles it by default.
- Removed Project Role from POC Check.
- Removed default End Date from Operations Post.
- Email notification from ToDo

### Chores

- Improved custom field/property setter/workflow/assignment rule creation process for developer.
- Updated email template for Employee ID change.
